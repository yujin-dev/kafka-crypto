# Kafka with Go
TCP로 수신되는 실시간 데이터 kafka producer, consumer

## Setup
```
go get gopkg.in/confluentinc/confluent-kafka-go.v1/kafka
```
or
```
go get -u github.com/confluentinc/confluent-kafka-go/v2/kafka
```
